Version 2.0.9 

  • Application

    • Update to .Net 4.7.2
    • Updated all dependencies to latest versions
    • Modpack import/export window now resizes columns to fit content
    • New option to set default modpack author in Options > Customize > Export Settings
      • Author will auto populate when creating a modpack
    • Changed default export to Autodesk Collada
      • This only affects new installs
    • Deleting ModPack from Mod List window now shows a wait message until it is complete
    • Updated all Skeleton files

  • Items

    • Weapons with gear as secondary will now display appropriate gear piece
      • This affects some two handed weapons (eg. Makai Fists, Sudarshana Chakra)

  • Bug Fixes

    • Fix for issue where some dae files would fail to import due to vertex colors
    • Fix for ModPack Wizard not changing to dark theme correctly
    • Fix for issue where modpack creator would crash when items with extra materials were present
    • Fix for issue where category saved when creating new materials was incorrect
    • Fix for issue where model export fails if bone name contains numbers
    • Fix for issue where diffuse with black alpha channel would appear incorrectly on model
